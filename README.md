
This project was completed as part of my **Biostatistics & Epidemiological Data Analysis using R** course at the University of Potsdam. \
It explores the KiGGS dataset (German Health Interview and Examination Survey for Children and Adolescents) through data cleaning, analysis, and a partial **meta-analysis** of selected health outcomes.

## Project Overview\
The project includes: \
- Cleaning and preprocessing the KiGGS dataset \
- Descriptive and exploratory statistical analysis \
- Conducting a **meta-analysis** on selected variables to synthesize findings across subgroups \
- Visualizing key outcomes and trends \
